# HKDF

HMAC-based Key Derivation Function (RFC 5869). SHA256. Permet de dériver des clés distinctes d'un même material via `info`. Labels RootWarden : `rootwarden-aes`, `rootwarden-totp`. Voir [[06_Securite/hkdf]].
