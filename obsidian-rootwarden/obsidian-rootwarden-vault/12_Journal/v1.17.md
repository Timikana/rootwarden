# v1.17.0 - 2026-04-25

Release de continuite : alignement schedules CVE/SSH audit, hygiene rotation cles API, detection agent Wazuh, qualite de vie utilisateur.

## Highlights

- **Multi-select serveurs sur les schedules CVE et SSH audit** - parite stricte entre les deux modules. UI nouvelle option "Plusieurs serveurs" avec grille de checkboxes + boutons Tout/Aucun + compteur live. Backend `target_type='machines'` + JSON array d'IDs. Migration [[08_DB/migrations/043_ssh_audit_schedules_machines_target]] aligne le schema SSH sur celui CVE.
- **Onboarding wizard 8/8** - bandeau felicitations vert/bleu avec CTA "Masquer definitivement" quand toutes les etapes sont validees, plutot que disparition silencieuse.
- **Banner rotation cle API legacy** - 2 niveaux sur `/adm/api_keys.php` (jaune info / rouge action requise) + rappel persistant compact sur le dashboard tant que `proxy-internal-legacy` coexiste avec une cle scopee.
- **Wazuh : `POST /wazuh/detect`** - bouton "Scanner" cyan dans l'onglet Deploiement pour detecter un agent installe hors RootWarden et l'enregistrer dans `wazuh_agents` sans reinstaller.
- **`scan_server_users` : auto-classification `nologin`** - le filtre awk excluait `nologin|false|sync|halt|shutdown`, ces comptes deviennent visibles dans l'inventaire et auto-classifies `excluded` (auditables sans polluer la liste des comptes geres).
- **Bug `/test_platform_key`** - check `platform_key_deployed` AVANT la connexion SSH pour eviter erreurs paramiko brutes sur un nouveau serveur ; message clair "Cle non deployee, clique Deployer d'abord".
- **`fmtLocalDate()` centralise** dans [[Code/www/js/utils.js]] charge depuis menu.php - retire la duplication dans [[Code/www/security/js/main.js]].
- **graph.json restaure depuis template** - Obsidian reecrit le fichier en permanence (scale, close, zoom), donc on versionne uniquement `graph.json.template` et `scripts/sync-obsidian-vault.py` recopie au premier checkout.
- **CI** - 2 fixes ruff (F821 import json manquant + F823 imports locaux qui shadowent le global). Pipeline a 11 jobs reste vert.
